Запуск скрипта:

1) Зарегестрироваться на сайте с погодой - https://openweathermap.org/ (Через VPN)
2) Получить API KEY 
3) Создать файл .env в папке с скриптом, вставить туда ключ в переменную WEATHER_API_KEY, а так же переменные подключения к БД
DB_USER DB_PASSWORD, взять их из своего подключения созданного в init.sql
4) Создать виртуальное окружение внутри папки с скриптом, перейти в консоли в папку с скриптом и запустить команду на создание виртуального окружения - python -m venv venv
5) Активировать виртуальное окружение - source venv/Scripts/activate
6) Установить зависимости pip install -r requirements.txt
