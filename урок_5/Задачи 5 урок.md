Задача 1: Меню с категориями
Создайте VIEW v_menu_with_categories, которое показывает все блюда с названием категории, ценой и статусом доступности. Отсортировать по категории и цене.

Задача 2: Сегодняшние бронирования
Создайте VIEW v_today_reservations, которое показывает все бронирования на текущий день с именем посетителя, временем и номером столика.

Задача 3: Выручка по категориям
Создайте VIEW v_revenue_by_category, которое показывает для каждой категории: название, количество проданных блюд, общую выручку и среднюю цену.

Задача 4: Топ-10 самых дорогих заказов
Создайте VIEW v_top_expensive_orders, которое показывает 10 самых дорогих заказов с именем посетителя, именем официанта и общей суммой.

Задача 5: Ингредиенты для конкретного блюда
Создайте VIEW v_dish_ingredients_summary, которое показывает для каждого блюда: название, количество ингредиентов, общий вес ингредиентов (сумма quantity_required) и список ингредиентов (через LISTAGG).

Задача 6: Статистика по посетителям
Создайте VIEW v_customer_statistics, которое показывает для каждого посетителя: ФИО, количество заказов, средний чек, дату последнего заказа, общую потраченную сумму.

Задача 7: Свободные столики на сегодня
Создайте VIEW v_free_tables_today, которое показывает номера столиков, на которые нет бронирований на текущий день.

Задача 8: Прибыльность блюд
Создайте VIEW v_dish_profitability, которое показывает для каждого блюда: название, цену, примерную себестоимость (сумма quantity_required * цена ингредиента) и маржинальность в процентах.

Задача 9: Ночные заказы (после 22:00)
Создайте VIEW v_night_orders, которое показывает заказы, сделанные после 22:00, с информацией о посетителе и официанте.

Задача 10: Рейтинг официантов
Создайте VIEW v_waiter_rating, которое ранжирует официантов по выручке (с использованием RANK() или ROW_NUMBER()).
