ЗАДАЧА 1: Функция для расчета возраста посетителя
Создайте функцию get_visitor_age, которая принимает ID посетителя и возвращает его возраст в годах. Если посетитель не найден, функция должна возвращать NULL.

ЗАДАЧА 2: Функция для проверки наличия блюда в заказе
Создайте функцию is_dish_in_order, которая принимает ID заказа и ID блюда, и возвращает TRUE (или 1), если блюдо есть в заказе, и FALSE (или 0), если нет.

ЗАДАЧА 3: Процедура для обновления цены блюда
Создайте процедуру update_dish_price, которая принимает ID блюда и новую цену, обновляет цену и выводит сообщение о количестве обновленных строк. Добавьте проверку, что цена положительная.

ЗАДАЧА 4: Процедура для удаления старых бронирований
Создайте процедуру delete_old_reservations, которая удаляет все бронирования старше указанного количества дней (параметр) со статусом 'cancelled' и выводит количество удаленных записей.

ЗАДАЧА 5: Функция для подсчета заказов посетителя
Создайте функцию count_visitor_orders, которая принимает ID посетителя и возвращает количество его заказов. Если посетитель не найден, возвращать 0.

ЗАДАЧА 6: Процедура для добавления нового ингредиента
Создайте процедуру add_ingredient, которая принимает название, единицу измерения и начальное количество, добавляет новый ингредиент и выводит его ID.

ЗАДАЧА 7: Функция для расчета общей суммы заказа
Создайте функцию get_order_total, которая принимает ID заказа и возвращает общую сумму заказа (суммируя quantity * price_at_moment из order_items).

ЗАДАЧА 8: Процедура для изменения статуса заказа
Создайте процедуру update_order_status, которая принимает ID заказа и новый статус, проверяет допустимость статуса (из списка 'new', 'preparing', 'served', 'paid', 'cancelled') и обновляет его.
