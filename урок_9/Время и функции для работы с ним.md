1) Date Хранит: дату и время с точностью до секунды
Формат по умолчанию: DD-MON-YY (зависит от NLS-настроек)
Диапазон: от 1 января 4712 г. до н.э. до 31 декабря 9999 г. н.э.
Размер: 7 байт

2) TIMESTAMP
Хранит: дату и время с дробными секундами
Точность: до 9 знаков после запятой (наносекунды)
Варианты:
TIMESTAMP — без часового пояса
TIMESTAMP WITH TIME ZONE — с часовым поясом
TIMESTAMP WITH LOCAL TIME ZONE — с локальным часовым поясом

3) INTERVAL
Хранит: промежутки времени
Типы:
INTERVAL YEAR TO MONTH — годы и месяцы
INTERVAL DAY TO SECOND — дни, часы, минуты, секунды

Функции для работы с временем

SYSDATE - функция для получения текущей даты и времени
CURRENT_DATE - Возвращает текущую дату и время в часовом поясе сессии
SYSTIMESTAMP - Возвращает текущую дату и время с дробными секундами и часовым поясом сервера
CURRENT_TIMESTAMP - Возвращает текущую дату и время с дробными секундами в часовом поясе сессии

Функции преобразования 
TO_DATE — строка → дата
TO_CHAR — дата → строка
TO_TIMESTAMP — строка → TIMESTAMP

Арифметика с датами 

Дата ± число = новая дата

SELECT SYSDATE + 1 FROM dual;           -- завтра
SELECT SYSDATE - 7 FROM dual;            -- неделю назад
SELECT SYSDATE + 30 FROM dual;           -- через месяц
SELECT SYSDATE + 0.5 FROM dual;          -- через 12 часов

Дата - дата = количество дней

SELECT 
    SYSDATE - hire_date AS days_employed,
    (SYSDATE - hire_date) / 365 AS years_employed
FROM waiters WHERE id = 1;

Дата + INTERVAL = Дата
SELECT SYSDATE + INTERVAL '1' DAY FROM dual;           -- завтра
SELECT SYSDATE + INTERVAL '2' HOUR FROM dual;          -- через 2 часа
SELECT SYSDATE + INTERVAL '30' MINUTE FROM dual;       -- через 30 минут
SELECT SYSDATE + INTERVAL '1-6' YEAR TO MONTH FROM dual; -- через 1 год 6 месяцев

Функции для извлечения частей даты 

EXTRACT - извлечение компонента 
Синтаксис
EXTRACT(interval from column)

Компонент	        Тип источника	                    Возвращает	        Диапазон
YEAR	            DATE,TIMESTAMP, INTERVAL YM	        Год	                1-9999
MONTH	            DATE, TIMESTAMP, INTERVAL YM	    Месяц	            1-12
DAY	                DATE, TIMESTAMP, INTERVAL DS	    День месяца	        1-31
HOUR	            TIMESTAMP, INTERVAL DS	            Час	                0-23
MINUTE	            TIMESTAMP, INTERVAL DS	            Минута	            0-59
SECOND	            TIMESTAMP, INTERVAL DS	            Секунда	            0-59.999999999

TRUNC округление до значение
Синтакси
TRUNC(column,value)

SELECT 
    TRUNC(SYSDATE, 'YYYY') AS first_day_of_year,
    TRUNC(SYSDATE, 'MM') AS first_day_of_month,
    TRUNC(SYSDATE, 'Q') AS first_day_of_quarter,
    TRUNC(SYSDATE, 'WW') AS first_day_of_week,
    TRUNC(SYSDATE, 'DAY') AS first_day_of_week_sun
FROM dual;
